### API Testing & Debugging

- **Postman** – GUI tool to create, test, and automate API requests (GET, POST, PUT, DELETE).
- **Insomnia** – Lightweight Postman alternative for REST & GraphQL.
- **Thunder Client** – API testing extension inside VS Code.
- **Hoppscotch** – Browser-based API testing tool (similar to Postman, fast).
- **curl** – CLI tool to send HTTP requests from the terminal.
- **HTTPie** – Human-friendly version of curl with cleaner syntax.
- **Swagger UI / OpenAPI** – Interactive API documentation & testing system.
- **GraphQL Playground** – Tool to test and explore GraphQL APIs.
- **ngrok** – Creates a public tunnel for your local server (great for testing webhooks).
- **Wireshark** – Network packet analyzer (deep inspection of traffic).

**Purpose:** Test, debug, analyze, expose APIs
