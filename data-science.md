## 🔧 Feature Engineering

| Concept | What it means |
|---------|--------------|
| **Encoding** | OneHot, Label, Target, Ordinal encoding for categorical features |
| **Scaling** | StandardScaler (zero mean), MinMaxScaler (0-1), RobustScaler (outlier-resistant) |
| **Imputation** | SimpleImputer (mean/median/mode), KNNImputer, IterativeImputer |
| **Feature selection** | RFE, correlation analysis, tree feature importance, SHAP values |
| **Cross-validation** | K-Fold, Stratified K-Fold, TimeSeriesSplit — prevent data leakage |
| **SMOTE / resampling** | Handle class imbalance — oversample minority, undersample majority |
| **Feature stores** | Feast, Tecton — centralized feature computation and serving |

## 📐 Statistics & Math

| Topic | Why it matters |
|-------|---------------|
| **Probability theory** | Distributions, Bayes theorem, conditional probability, expectation/variance |
| **Linear algebra** | Matrices, vectors, eigenvalues, SVD — core of neural network math |
| **Calculus** | Derivatives, chain rule, partial derivatives — backpropagation math |
| **Hypothesis testing** | p-values, confidence intervals, Type I/II errors, power analysis |
| **A/B testing** | Statistical significance, sample size calculation, multiple testing |
| **Information theory** | Entropy, cross-entropy, KL divergence — loss functions |
| **Optimization** | Gradient descent variants: SGD, Adam, AdaGrad, RMSProp |
| **Bayesian statistics** | Prior/posterior, MCMC, probabilistic programming |
